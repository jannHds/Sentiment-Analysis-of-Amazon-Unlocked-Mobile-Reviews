# Sentiment Analysis of Amazon Unlocked Mobile Reviews

## Project Overview
This project applies Natural Language Processing (NLP) techniques to analyze customer reviews from the Amazon Unlocked Mobile dataset. The system automatically classifies reviews as **Positive**, **Neutral**, or **Negative** using the **TextBlob** sentiment analysis library.

The goal is to help businesses quickly understand customer opinions and identify strengths and weaknesses in their products without manually reading thousands of reviews.

## Dataset
The project uses the **Amazon Unlocked Mobile Reviews dataset**, which contains thousands of customer reviews for mobile phone products.

Each review may include:
- Product name  
- Customer rating  
- Review text  

---

## Technologies Used
- Python  
- TextBlob (Sentiment Analysis)  
- Pandas (Data Processing)  
- Matplotlib (Data Visualization)  
- Jupyter Notebook  

---

## Project Workflow
1. Load the Amazon mobile reviews dataset.
2. Identify the **top 10 products with the highest number of reviews**.
3. Select one product for analysis.
4. Apply **TextBlob sentiment analysis** to calculate the polarity score of each review.
5. Classify reviews into:
   - Positive
   - Neutral
   - Negative
6. Visualize the sentiment distribution using charts.

---

## Results
The system analyzes customer opinions and visualizes the sentiment distribution for selected products. Most popular products receive predominantly positive reviews, while a smaller percentage are neutral or negative.
